# 🌤️ WeatherView - Modern Weather App

WeatherView is a simple and modern weather application that shows real-time weather data, air quality, and other details. Users can search for any city or use their current location to get accurate weather information.

---

## 🚀 Features

- 🌡️ Real-time weather data  
- 🌍 Search weather by city name  
- 📍 Get weather using current location  
- 💨 Wind speed and direction  
- 💧 Humidity and pressure  
- 🌫️ Air Quality Index (AQI)  
- 🌅 Sunrise and sunset information  
- 🎨 Clean and modern UI with animations  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- WeatherAPI
